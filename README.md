# 旅行记录网站

一个用于记录铁路和飞机旅行的个人旅行记录网站，支持登录注册、详细的旅行信息管理和可视化的旅行轨迹地图。

## 功能特性

### 核心功能
- **用户系统**：登录、注册功能
- **旅行记录**：添加、查看、删除旅行记录
- **交通方式**：支持铁路和飞机两种交通方式
- **详细信息**：
  - 铁路：国家、类型（动车/普速）、车系、车型
  - 飞机：地区、航空公司、机型
- **旅行轨迹**：在世界地图上显示旅行轨迹，支持多次往返显示
- **地图功能**：标记世界各国首都和中国各省会城市

### 技术特性
- **响应式设计**：适配不同屏幕尺寸
- **数据持久化**：使用localStorage存储数据
- **现代化界面**：渐变背景、卡片式设计、流畅的交互效果
- **离线支持**：基本功能可离线使用

## 技术栈

- **前端框架**：React 18
- **地图库**：Leaflet
- **样式**：CSS3
- **图标**：Font Awesome
- **数据存储**：localStorage

## 快速开始

### 直接打开
1. 找到 `travel-app/index.html` 文件
2. 双击该文件，在默认浏览器中打开
3. 注册一个新账号或使用已有账号登录

### 使用本地服务器（推荐）
1. 打开终端，导航到项目目录：
   ```bash
   cd travel-app
   ```
2. 启动本地服务器：
   - 使用Python：
     ```bash
     python -m http.server 8000
     ```
   - 使用Node.js：
     ```bash
     npx http-server -p 8000
     ```
3. 在浏览器中访问 `http://localhost:8000`

## 使用指南

### 1. 注册/登录
- 首次使用需要注册一个账号
- 密码要求至少8位，包含数字和字母
- 注册后自动登录

### 2. 添加旅行记录
1. 在「旅行记录」页面点击「添加旅行记录」
2. 选择交通方式（铁路/飞机）
3. 选择出发地和目的地（省份 → 城市）
4. 选择日期
5. 根据交通方式填写详细信息：
   - **铁路**：选择国家、类型、车系、车型
   - **飞机**：选择地区、航空公司、机型
6. 点击「保存记录」

### 3. 查看旅行记录
- 在「旅行记录」页面查看所有已添加的旅行记录
- 每条记录显示交通方式、路线、日期和详细信息
- 可点击删除按钮删除不需要的记录

### 4. 查看旅行轨迹
- 在「旅行轨迹」页面查看世界地图
- 地图上显示：
  - 世界各国首都（灰色小圆点）
  - 中国各省会城市（红色小圆点）
  - 旅行轨迹（曲线，铁路为蓝色虚线，飞机为红色实线）
- 多次往返同一路线时，轨迹会显示为不同的曲线，避免重叠

## 数据说明

- 所有数据存储在浏览器的localStorage中
- 包含以下数据：
  - 用户信息：用户名、密码
  - 旅行记录：交通方式、路线、日期、详细信息
- 清除浏览器缓存会导致数据丢失，请谨慎操作

## 注意事项

1. 地图功能需要网络连接才能加载底图
2. 数据存储在浏览器本地，换设备或清除缓存会丢失数据
3. 密码以明文存储在localStorage中，仅用于本地测试
4. 本项目仅用于个人记录，不支持多用户或云端同步
